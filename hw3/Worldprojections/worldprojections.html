In this project I learned how to display images in different projections

I displayed the map in different projections using QGIS by using the map that was given to me and dropping it in QGIS. Then I chose different Point Coordinate Reference Systems (CRS) located in the bottom right corner of the QGIS page. The projections showed different bubbles from the Indicatrix Mapper plugin that I installed and added to the project.

WGS84 Projection

Some observations: Cylindrical map with conformal projections and vertal stretch.

Aitoff Projection

Some observations: Equal-area map projection (preserves area) with an ellipse shape. Direction is not preserved accurately.

Psuedo Mercator Projection

Some observations: Cylindrical map with conformal projections and vertal stretch.

Sphere Winkel Projection

Some observations: Cylindrical map with equal-area projections.

Cylindrical Equal Area Projection

Some observations: Cylindrical shape projection with equal area projections. The meridian and latitude lines are equally spaced (make a square).

Equidistant Conic Projection

Some observations: Cone shape map with distance preservations. Meridians are equally spaced from the apex of the cone. The shape and area are distorted.

North Pole Azimuthal Equidistant

Some observations: The North Pole is the central point of the projection. It preserves distance from the North Pole.

Sphere Polyconic

Some observations: Conic projection with small-area coverage and conformal properties.

World Polyconic

Some observations: Uses multiple conic projections.
